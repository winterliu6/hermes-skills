# Hermes Skills

[Hermes Agent](https://hermes-agent.nousresearch.com) 工作流技能库，包含内容创作与自动化工作流。

## 内容

### content-studio
小红书/抖音/公众号内容生成器。支持AI Agent教程、安全急救科普、技术教程等多种内容方向，自动产出完整内容包（标题+正文+分镜+配图方案）。

适用场景：
- 公众号文章（800-1500字，含干货+故事钩子+口诀）
- 小红书知识卡片（6-9图流）
- 抖音视频脚本（30-60秒口播+分镜）
- 技术教程（代码实战+踩坑记录）

### security-newsletter
企业网络安全意识宣贯邮件自动化流水线。每周自动采集安全情报、生成并发送安全意识邮件。

适用场景：
- IT部门每周安全宣贯
- 定期威胁情报汇总
- 企业安全意识培训

## 使用方法

将 SKILL.md 放入 Hermes Agent 的 skills 目录即可使用。详细说明请参考各目录下的 SKILL.md。
