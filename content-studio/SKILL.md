# 内容工坊 (Content Studio)

## 用途
给一个主题，自动生产一套完整的内容包，不用对方去想文案。

## 平台与格式

| 平台 | 输出格式 | 字数 | 配图 |
|------|---------|------|------|
| 小红书图文 | 标题+正文+6-8格分镜+标签+配图建议 | 300-500字 | 漫画/实拍 |
| 抖音视频脚本 | 口播文案+分镜+表情提示 | 以30-60秒为单元 | 视频画面描述 |
| 公众号文章 | 标题+故事钩子+干货分段+口诀+CTA | 800-1500字 | 封面+4-6张内文图 |

## 使用方式
对 Hermes 说：`用 content-studio，主题是XXX，平台是[小红书/抖音/公众号]`

## 通用工作流（每次必做）

1. **先调研，后动笔** — 搜索相关关键词在目标平台已有的内容
2. **确定角度** — 选流量最大、竞品最少的切入点
3. **写内容**
4. **配图方案** — 封面图+内文图的获取方式
5. **发布建议** — 发布时间、转发话术

## 公众号文章

### 爆款标题公式
```
[权威主语]紧急提醒：[场景痛点]别做X！[N个动作]一定要记住
```

### 内容结构模板（安全/科普类）
1. **故事钩子** — 真实案例 + 当事人做法 + 权威评价
2. **痛点数据** — 1-2个数字让读者觉得"这事和我有关"
3. **核心干货** — 编成口诀，分步展开，✅做什么 → ❌不做什么 → 为什么
4. **误区对比表** — ❌土方法 → 为什么危险 → ✅正确做法
5. **顺口溜/口诀** — 押韵、短、易记
6. **转发引导（CTA）** — "转发到家族群"、"给爸妈看"

### 适老化排版要求
- 字号18-20px，行间距1.75-2.0倍
- 每段不超过3行，关键数字加红加粗
- 多用列表、表格、图标，少用长段落

### 配图方案
- **封面图**：AI绘图prompt
- **步骤示意图**：卡通科普风格
- **口诀图/对比图**：编辑器直接做文字图

AI绘图prompt模板：`[场景描述]，[人物关系]，[风格]，[背景]，[色调]`

## 安全急救类内容（适老化方向）

### 选题优先级

| 优先级 | 方向 | 流量 | 竞争 |
|:------:|:----|:----:|:----:|
| 🥇 | 安全急救类（烫伤/卡喉/跌倒） | 高 | 中低 |
| 🥈 | 观念冲突类（科学vs传统） | 中 | 中 |
| 🥉 | 游戏/互动类 | 低 | 低 |

### 内容设计原则（硬约束）
1. **字体最大** — 文字量减半，字号至少大两倍
2. **一句话一个意思** — 不分段嵌套
3. **图片为主** — 每页至少60%是图
4. **操作步骤 ≤ 3步** — 超过3步就拆成多张图
5. **零抽象术语** — "浏览器"→"上网图标"
6. **正向引导** — 少说"不要做X"，多说"可以这样做Y"
7. **对比参考** — 配"正确做法 vs 错误做法"对比图
8. **编口诀/顺口溜**

## 公众号自动发布（微信API方式）

当用户提供公众号AppID和AppSecret后，可通过微信API自动发布文章。

### 前置条件
1. 开通公众号开发者模式
2. 拿到AppID和AppSecret
3. 将服务器IP加入IP白名单
4. 凭证本地保存，不入skill正文

### API流程

```
1. token获取: POST /cgi-bin/token?grant_type=client_credential&appid=APPID&secret=APPSECRET
   → access_token（有效期2小时）

2. 图片上传: POST /cgi-bin/material/add_material?access_token=TOKEN&type=image
   → media_id + url（CDN图床地址）

3. 创建草稿: POST /cgi-bin/draft/add?access_token=TOKEN
   Body: {"articles": [{"title": "...", "content": "<section>HTML内容</section>",
     "thumb_media_id": "封面图media_id", "need_open_comment": 1, "show_cover_pic": 1}]}
   → media_id（草稿ID）

4. 发布: POST /cgi-bin/freepublish/submit?access_token=TOKEN
   Body: {"media_id": "草稿ID"}
   → 仅已认证公众号可用
```

### 关键限制

| 步骤 | 个人订阅号（未认证） | 已认证 |
|:----|:------------------:|:-----:|
| 获取token | ✅ | ✅ |
| 上传素材 | ✅ | ✅ |
| 创建草稿 | ✅ | ✅ |
| 自动发布 | ❌ errcode 48001 | ✅ |

**个人订阅号：草稿创建成功后提示用户手动去后台点"发布"。**

### 图片嵌入规则
文章HTML中的图片必须使用上传接口返回的完整CDN URL。

### 已知技术坑
1. execute_code和terminal文件系统不共享，文件操作必须在terminal中完成
2. access_token有效期2小时，每次使用前重新获取
3. 服务器IP可能变化，需在公众号后台更新白名单
